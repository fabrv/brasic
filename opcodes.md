# OPCODES
<>+-[].,#:
* **(LFT)** LEFT *number*
* **(RHT)** RIGHT *number*
* **(ADD)** ADD *number*
* **(SUB)** SUBSTRACT *number*
* **(WHL N)** WHILE NOT ZERO
  * **(WHL 0)** WHILE ZERO
  * **(END)** END
* **(PRT)** PRINT
* **(INP)** INPUT
* **(IFN, THN, ELS)** IF-THEN-ELSE
* **(MOV)** MOVE
* **(CPY)** COPY
* **(CLR)** CLEAR
